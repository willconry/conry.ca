Grocery Assistant

Available on GitHub

An Android application that assists visually impaired people with grocery shopping using indoor navigation and voice communication. Indoor positioning is achieved using trilateration with distances approximated from the signal strength of Bluetooth beacons. Navigation uses Navigine's API. The voice interface uses AWS Lex. Also has a shopping list feature that stores the name and location of grocery items using AWS DynamoDB and AWS Lambda.
